documentation.md
# Documentation Bundle

bundle:
  name: documentation
  version: 1.0.0

includes:
  - bundle: foundation

tools:
  - filesystem
  - web_search
  - grep
  - glob

behaviors:
  - technical-writing
  - research

context:
  include:
    - docs/README-TEMPLATE.md

Bundles are YAML + markdown. Combine providers, tools, and behaviors into reusable capabilities you can customize.

What you're seeing

Bundle selection

Each bundle is a complete capability. Switch between them to see different configurations and use cases.

Transparent configuration

The YAML shows exactly what the bundle includes. No black box. You can read, modify, and create your own.

Real work

These bundles do actual work—create documentation, review code, build presentations. Not toy demos.
